## 회사 개요

카트만두 소재 네팔 인력송출(해외고용) 회사. 레포 원본 **LICENSE PDF**(네팔 노동교통관리부/해외고용국 DoFE 발급, 2004-10-04자)로 해외고용 라이선스 **No. 69/052/053**가 1차 확인된다 — 단 이 원본에는 상호가 복수형 `Sunkoshi Manpower Services Pvt. Ltd.`로 인쇄되어 있고 주소는 없다. 일본 OTIT 「네팔 인정 송출기관(技能実習)」 명단에 **NPL003184**로 등재(주소 Kathmandu-30 Maitidevi, 이사 4인)되어 일본 채널 송출 자격이 일본 준정부 문서로 교차확인된다. 자사 사이트 기준 1995년 설립, 걸프·말레이시아·일본·한국 대상 송출을 표방하나, 설립연도·법인등록번호·훈련 연계·한국 실적은 정부 등기로 확인되지 않는다.

네팔→일본 SSW(介護·宿泊)·한국(E-7/E-8) 관점에서 정우인재개발원의 **송출(인력송출사) 파트너** 그룹. **DoFE 해외고용 라이선스(레포 1차 문서)** + **OTIT 인정 송출기관(NPL003184)** 등재가 근거다. 다만 OTIT 등재는 技能実習(TITP) 기준이므로 特定技能(SSW/介護·宿泊) 송출 자격·실적은 별도 확인이 필요하고, 자사의 **한국 EPS 송출 '854개사 중 5위'** 주장은 자사 마케팅 문구로 교차검증되지 않는다.
