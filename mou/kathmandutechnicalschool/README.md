# Kathmandu Technical School (KTS) 조사 보고서

조사일: **2026-07-18**  
대상: [Kathmandu Technical School 공식 웹사이트](https://ktsnepal.com/)  
목적: 정우인재개발원 홈페이지 MOU 파트너 정보와 개호 교육 카탈로그 제작을 위한 사실 확인

## 결론 요약

KTS는 네팔 카트만두 수케다라에 위치한 직업훈련학교로, 공식 사이트에서 **개호·고령자 돌봄(Health Care)**과 **호스피탈리티(Hospitality Management)** 과정을 운영한다고 안내한다. 개호 분야의 핵심 공개 과정은 다음 두 가지다.

1. **Caregiver and Aged Care**: 3개월, 총 390시간의 단기 직업훈련 과정
2. **Certificate III in Individual Support (Ageing and Disability)**: KTS가 호주 CIBT와의 협력 과정으로 소개하는 52주 과정

카탈로그에서 가장 근거가 탄탄한 내용은 KTS 공식 페이지가 직접 연결한 **CTEVT 「Care Giver, Short Term Curriculum, Second Revision 2079 (2022 AD)」**이다. 총 과정시간은 390시간으로 일관되지만, 원문의 이론·실습 합계와 프로젝트 시수에는 서로 맞지 않는 표기가 있어 아래에 별도로 표시했다. 따라서 카탈로그에는 **390시간 과정**과 학습영역은 사용할 수 있지만, 이론·실습 비율과 프로젝트 154시간은 KTS 또는 CTEVT의 서면 확인 전까지 확정 수치로 홍보하지 않는다. 또한 교육기관 수료증과 **NSTB Level 1 기능시험 합격**은 별도 절차다.

### 시수 검증상 중요한 불일치

| 확인 지점 | 원문 표기 | 판단 |
|---|---:|---|
| 전체 과정 | 390시간 | 구조표의 모든 모듈 총시간 합계와 일치 |
| 구조표 머리·마지막 합계 | 이론 67 + 실습 323 | 행별 숫자와 2시간 차이 |
| 모듈 행별 합산 | 이론 65 + 실습 325 | 각 행을 직접 합산한 값 |
| 프로젝트 구조표 | 실습 154시간 | 전체 390시간에 포함 |
| 프로젝트 상세 페이지 | 이론 1 + 실습 9 = 10시간 | 154시간 표기와 충돌 |

이 때문에 공개 카탈로그에는 “390시간 실무 중심 과정”이라고 쓰는 것이 안전하다. “실습 323시간”, “실습 83%”, “154시간 OJT”는 KTS의 실제 운영 시간표 또는 CTEVT 정오표를 받은 뒤 확정한다.

### 통합 프로젝트의 세부 수행항목

- **영양·식사 6개 과업:** 고형식 조리·급식, 연식 조리·급식, 유동식 조리·급식
- **개인·환경 위생 7개 과업:** 구강, 눈, 모발, 손발톱, 피부 위생, 등 마사지, 대상자 생활공간 청결
- **배설 4개 과업:** 소변·대변 관리, 기저귀 사용, Ezevac Enema 보조, 객담 관리
- **안위 3개 과업:** 방 정돈, 침상 만들기, 체위 관리
- **공통 작업:** 실습장 준비, PPE 사용, 사용 도구 세척·보관, 작업장 정리, 수행기록 작성

Care Home 등 실제 작업현장 방문은 CTEVT의 훈련기관 운영기준에 별도로 제시되며, 위 통합 프로젝트 154시간과 동일한 개념으로 단정할 수 없다.

## 과정 2 — Certificate III in Individual Support

KTS는 이 과정을 **Ageing and Disability**, 52주 과정으로 소개하며 호주 **Crown Institute of Business and Technology (CIBT)**와의 협력을 표기한다. CIBT 공식 페이지는 국가과정코드 **CHC33021**, CRICOS **114133F**, 기간 **52주**, 현장실습 **120시간**으로 안내한다.

KTS 공개 페이지의 15개 유닛은 다음과 같다.

| 코드 | 학습 유닛 |
|---|---|
| CHCCCS031 | 개인별 맞춤 지원 제공 |
| CHCCCS038 | 지원 대상자의 역량강화 촉진 |
| CHCCCS040 | 독립성과 웰빙 지원 |
| CHCCCS041 | 건강한 신체 시스템 인식 |
| CHCCOM005 | 보건·지역사회 서비스 의사소통 |
| CHCLEG001 | 법적·윤리적 기준에 따른 업무 |
| CHCDIV001 | 다양한 배경의 사람들과의 업무 |
| HLTWHS002 | 직접 돌봄 업무의 안전한 작업 관행 |
| HLTINF006 | 감염 예방·관리 절차 적용 |
| CHCAGE011 | 치매 대상자 지원 |
| CHCAGE013 | 고령자 돌봄 서비스 제공 |
| CHCPAL003 | 완화의료 접근에 따른 대상자 지원 |
| CHCDIS011 | 강점 기반 기술개발 지원 |
| CHCDIS012 | 지역사회 참여와 사회적 포용 지원 |
| CHCDIS020 | 장애인 지원 서비스 제공 |

> **게시 전 필수 확인:** CIBT 공식 페이지의 52주·120시간 현장실습은 호주 등록과정 정보다. KTS에서 네팔 현지 교육을 어떤 방식으로 제공하는지, 평가기관과 최종 자격증 발급기관이 어디인지, 네팔 수강생에게 동일한 CHC33021 자격이 발급되는지는 KTS–CIBT 계약서와 학생 안내서로 확인해야 한다. 확인 전에는 “호주 공인 자격증을 네팔에서 취득”이라고 단정하지 않는다.

- 기본 건강관찰: 체온계, 혈압계, 혈당측정기, 산소포화도계
- 호흡·보조: 네뷸라이저, 산소마스크·비강캐뉼라, 가습기, 폐활량계
- 개인위생·배설: 베드팬, 기저귀, 물티슈, 손소독제, 고무시트, 객담용기
- 감염·안전: 장갑, 마스크, 앞치마, 보안경, 작업복, 소화기, 구급함
- 영양·식사: 냉장고, 조리대·싱크대, 가열기기, 조리도구, 급식컵

> 장비 사진 옆에는 “CTEVT 커리큘럼 기준 실습장비”라고 표기하고, KTS 현장 실사로 실제 보유가 확인된 품목만 “KTS 보유 장비”라고 표현한다.

## 제3자 검증 신호

- 네팔 NSTB 공개 기능시험 결과에서 **Kathmandu Technical School Pvt. Ltd.**가 Caregiver (Personal & Home Care Aid), Level 1 시험센터로 여러 차례 기재된다.
- 이는 KTS에서 개호 기능시험이 실제 운영된 정황을 보여주지만, 모든 KTS 수료생의 합격이나 자동 자격 취득을 의미하지 않는다.
- KTS LinkedIn은 개호 실습 연계기관과 취업연계 사례를 안내하지만 계약서나 현재 유효성은 별도 확인이 필요하다.

## 카탈로그 표현 가이드

### 사용 가능한 표현

- “3개월·390시간의 Caregiver/Aged Care 실무과정”
- “총 390시간의 실무 중심 Caregiver/Aged Care 과정”
- “개인위생, 영양, 배설, 이동·안위, 건강장비, 심리사회적 지원을 통합 학습”
- “과정 수료 후 자격요건을 갖춘 수료생은 NSTB Level 1 기능시험에 응시 가능”
- “KTS가 CIBT와의 협력과정으로 소개하는 52주 Certificate III 과정”

### 확인 전 사용하면 안 되는 표현

- “수료만으로 NSTB 국가자격 자동 취득”
- “일본·한국에서 자동 인정되는 국제 개호면허”
- “일본 특정기능 개호시험과 일본어시험이 면제되는 과정”
- “KTS에서 호주 CHC33021 자격증을 확정 발급”
- 현장 확인 없이 “모든 최신 개호장비를 완비”

일본 특정기능 1호 개호 취업은 교육 수료와 별개로 일본 측 제도에 따른 일본어 및 개호 관련 시험, 고용계약, 재류자격 절차를 충족해야 한다.
