## 일본·한국 송출 관련성

네팔→일본 SSW(介護·호텔)·한국(E-7)을 지향하는 인력송출사로, 정우인재개발원의 **송출(파견원)** 그룹. OTIT 송출기관 명단(NPL003166) 등재로 일본 송출 채널 실재가 정부계 자료로 확인되나, 해당 명단은 技能実習(TITP) 기준이므로 MOU가 명시한 特定技能(SSW) 介護·호텔 인가·실적은 별도 확인이 필요하다. 자사 웹사이트는 일본·한국을 송출 대상국으로만 열거할 뿐 介護·호텔·E-7 같은 SSW 세부 트랙은 명시하지 않는다.
